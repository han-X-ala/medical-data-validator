# Medical Data Validator

This project validates medical records using Python and regex.

## Features
- Validates patient ID format using regex
- Checks age (must be >= 18)
- Validates gender values
- Ensures medications list contains only strings
- Detects invalid records and prints detailed errors

## Example Output
Unexpected format 'patient_id: p1002' at position 1.

## Technologies Used
- Python
- Regex (re module)

## How to Run
1. Clone the repository
2. Run the Python file:
   python filename.py
